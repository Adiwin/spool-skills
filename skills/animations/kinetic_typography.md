# Pattern Skill: Kinetic Typography

Use when the message itself is the visual hero.

## Recipe
- 1 to 3 text blocks max
- large type
- strong alignment system
- word or line staggers instead of many decorative objects

## Motion
- prefer `textIn`
- use `slide`, `slideAndMask`, `blurIn`, or `blurSlideIn`
- secondary accents should be minimal shapes or bars

## Avoid
- mixing too many text animation styles in one beat
- letter-level chaos unless explicitly asked
