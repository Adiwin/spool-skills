# Pattern Skill: Stagger Cascade

Use for lists, features, card grids, and multi-item reveals.

## Recipe
- keep all items on one shared motion system
- stagger by 40 to 100ms
- each item should use a small move + opacity or small scale + opacity

## Rule
The more items there are, the smaller each item’s motion should be.

## Best For
- benefits sections
- pricing cards
- feature bullets
- dashboard widgets
