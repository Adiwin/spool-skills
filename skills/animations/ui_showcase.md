# Pattern Skill: UI Showcase

Use for dashboards, app promos, SaaS explainers, and interface reveals.

## Composition
- main frame or app card
- 1 to 3 supporting callouts
- background surface kept quiet

## Motion Recipe
- main UI enters with slide + fade
- callouts stagger in
- small emphasis scale on active card only

## Tone
Precise, modern, and sharp. Motion should feel engineered, not flashy.
