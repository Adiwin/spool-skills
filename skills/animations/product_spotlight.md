# Pattern Skill: Product Spotlight

Use for a single product render, package, or object-centered promo.

## Recipe
- one hero object
- optional pedestal/card behind it
- short supporting text lockup

## Motion
- hero object: `grow_in_group` or subtle blur-to-sharp reveal
- support copy: delayed `textIn`
- accents: tiny move or rotate only if they help focus

## Tone
Premium, centered, and composed.
